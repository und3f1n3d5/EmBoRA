# EUMAS 2026 camera-ready package

This folder contains the corrected camera-ready package for the accepted EUMAS 2026 full paper:

**Emotionally Bounded-Rational Agents in Repeated Games: An Interpretable Dual-System Architecture and Diagnostic State-Stability Study**

The package includes:

- LaTeX source: `eumas2026_article_camera_ready.tex`
- PDF: `eumas2026_article_camera_ready.pdf`
- Bibliography: `eumas2026_references_camera_ready.bib` and `eumas2026_article_camera_ready.bbl`
- LNCS files: `llncs.cls`, `splncs04.bst`
- Figures: `figures/`
- Tables and generated CSV/TeX/Markdown summaries: `tables/`
- Reproducibility/code materials: `github_materials/`
- Experiment documentation and manifests: `experiment_docs/`, `experiment_manifest.json`
- GitHub upload guide: `POWERSHELL_GITHUB_SUBMISSION_GUIDE.md`
- Russian changelog: `camera_ready_changelog_ru.md`

## Final correction note

The architecture overview figure has been restored to the submitted implementation-level diagram. The package no longer uses the simplified linear schematic as Figure 1. The text around Figure 1 was adjusted so that the caption and model description match the restored diagram.

The text was also re-read with a final editorial pass focused on density, concrete positioning, and removal of overly defensive or generic phrasing. No new scientific claims, code changes, or experiments were introduced.
